Real-time environmental safety system using NOAA Heat Index formula with 5-level danger classification.  
**Tech:** React 18 · Java 17 · Spring Boot 3 · MongoDB · Docker · Nginx

Intelligent pharmacy platform with AI micro-agents (ExpiryAgent, LowStockAgent, AutoResolveAgent).
